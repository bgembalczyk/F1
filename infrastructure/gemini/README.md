# infrastructure/gemini

Pakiet integruje klienta Gemini i pomocnicze komponenty infrastrukturalne.

## Public API
- `infrastructure.gemini.GeminiClient`
- `infrastructure.gemini.ModelConfig`
- `infrastructure.gemini.ModelState`
- `infrastructure.gemini.DEFAULT_MODELS`
- `infrastructure.gemini.DEFAULT_TIMEOUT`

## Internal
Pozostałe moduły (`cache.py`, `orchestration.py`, `transport.py`, `response_parser.py`, `model_selector.py`) są internal i nie są częścią stabilnego API.
