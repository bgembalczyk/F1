# config/

Katalog konfiguracyjny – pliki z kluczami API i innymi wrażliwymi danymi.

## Klucz Gemini API

Utwórz plik `config/gemini_api_key.txt` i wpisz do niego swój klucz API Gemini
(tylko klucz, bez spacji ani nowych linii):

```
AIza...twój_klucz...
```

Plik ten jest wykluczony z repozytorium przez `.gitignore`.

Klucz API możesz wygenerować na stronie:
https://aistudio.google.com/app/apikey
